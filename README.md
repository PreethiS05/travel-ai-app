# 🌍 AI Travel Planner Pro

A production-grade AI-powered travel planning web application — like MakeMyTrip / Booking.com — with intelligent recommendations, Firebase Google Authentication, and a premium booking-style UI.

![Built with Next.js](https://img.shields.io/badge/Next.js-14-black) ![Tailwind](https://img.shields.io/badge/Tailwind-CSS-blue) ![Firebase](https://img.shields.io/badge/Firebase-Auth-orange)

---

## ✨ Features

- 🔐 **Google Authentication** via Firebase
- 🧠 **Smart Recommendation Engine** — weighted scoring (Mood × 4, Activity × 3, Type × 2, Budget × 1)
- 🗣️ **NLP Query Parsing** ("cheap beach for honeymoon" → parsed filters)
- 💰 **Budget Logic** (Low ≤ ₹1500, Medium ≤ ₹4000, High = unlimited)
- 🏨 **30+ Realistic Destinations** with hotels, pricing, ratings, and images
- ⚡ **Real-time Feel** — simulated loader + skeleton cards
- 🏆 **Best Budget Match** highlight
- 📊 **Estimated Trip Cost** (2 nights)
- 📱 **Fully Responsive** — mobile-first design
- 🎨 **Premium UI** — glassmorphism, gradients, smooth animations
- 🚫 **Graceful Fallback** — handles no-match cases beautifully
